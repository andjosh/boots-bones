Boots & Bones was built to fill a need.

Josh needed the ability to combine the bare-naked building blocks found in Skeleton's 16-column grid with the fluid grid found in Bootstrap. He also wanted to build another step outward in the responsive design, allowing for an expansion into a 1280px framework. The world is gettin' bigger, yo.

Boots & Bones combines lightweight grid-kindling with fluidity and big-screen dreams.

About Bootstrap

Built at Twitter by @mdo and @fat, Bootstrap utilizes LESS CSS, is compiled via Node, and is managed through GitHub to help nerds do awesome stuff on the web.

Docs & Support

The easiest way to really get started with Bootstrap is to check out the full docs and info at twitter.github.com/bootstrap. A 12-column responsive grid, dozens of components, JavaScript plugins, typography, form controls, and even a web-based Customizer to make Bootstrap your own.

Three Core Principles

Boots & Bones is built on three core principles (which should be familiar by now):

  • A Responsive Grid Down To Mobile: Scaling from a browser to tablets to mobile.
  • Fast to Start: It's a tool for rapid development with best practices
  • Style Agnostic: It provides the most basic, beautiful styles, but is meant to be overwritten. Seriously, you shouldn't be using this kind of thing on its own.

About Skeleton

Skeleton is a small collection of well-organized CSS files that can help you rapidly develop sites that look beautiful at any size, be it a 17" laptop screen or an iPhone. It's based on a responsive grid, but also provides very basic CSS for typography, buttons, forms and media queries. Go ahead, resize this super basic page to see the grid in action.

Docs & Support

The easiest way to really get started with Skeleton is to check out the full docs and info at www.getskeleton.com. Skeleton is also open-source and has a project on git, so check that out if you want to report bugs or create a pull request.
